# 卷积神经网络

## 动机

全连接网络会忽略图像的局部结构与空间关系。CNN 用卷积和池化把局部模式提取纳入模型结构。

## 仓库中的实现

- `ConvolutionLayer`：支持朴素卷积和基于 `im2col` 的高效实现。
- `PoolingLayer`：支持最大池化及其反向传播。
- `libs/util.py`：提供 `im2col` / `col2im`。
- `t21-简单的卷积网络实现.ipynb`：实现 `SimpleConvNet` 并训练 MNIST。

## `SimpleConvNet` 结构

`conv -> relu -> pool -> affine -> relu -> affine -> softmax`

它体现了鱼书 CNN 章节的最小闭环：

- 卷积层负责局部特征提取。
- 池化层负责降采样。
- 全连接层负责最终分类。

## 数据形态变化

- 全连接网络：通常输入 `(N, 784)`。
- CNN：输入 `(N, 1, 28, 28)`。
- 这也是 `load_mnist(flatten=False)` 存在的直接原因。
