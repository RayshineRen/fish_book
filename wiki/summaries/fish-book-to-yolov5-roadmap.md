# 鱼书到 YOLOv5 的实践路径

## 来源文件

- `raw/docs/【零基础两个月速成】计算机考研复试项目准备-爱代码爱编程.md`

## 这份资料在本库中的定位

- 它不是鱼书内容本身。
- 它提供了一条面向复试项目的实践路线：环境配置 -> 学鱼书 -> 调试 YOLOv5 -> 包装项目成果。
- 因此它应被视为“鱼书之外的应用外延”，而不是核心理论来源。

## 路线概览

1. Windows 下配置 Python / Anaconda / PyTorch / CUDA 等环境。
2. 用鱼书理解前向传播、反向传播、梯度下降、特征提取等基础概念。
3. 调试 YOLOv5 目标检测项目。
4. 把调试过程包装成可以展示工作量、学习能力和科研意识的项目经历。

## 对鱼书学习的启发

- 这份资料把鱼书放在一个很实际的位置上：不是为了停留在“从零实现”，而是为了给后续读框架代码和调试工程项目打底。
- 它提醒了一个现实问题：初学者在图像 IO、权重文件、环境兼容性上的痛点，往往比公式本身更早成为障碍。
- 它也说明鱼书最有价值的地方，是帮助建立训练流程的整体感，而不是直接产出工业级模型。

## YOLOv5 部分包含什么

- 数据集结构的直观解释，例如 `images/` 与 `labels/` 的对应关系。
- `detect.py` 常用参数的解释，例如 `--img-size`、`--conf-thres`、`--iou-thres`、`--save-txt`、`--augment`、`--exist-ok`。
- `train.py` 训练产物的说明，例如 `best.pt`、`last.pt`、`hyp.yaml`、`results.txt`。

## 应该如何使用这份资料

- 如果你的目标是学懂鱼书主体，这页只需要当背景材料看。
- 如果你的目标是把鱼书知识迁移到工程项目，这页值得和 [[concepts/学习地图|学习地图]] 一起看。
- 它适合作为“下一步做什么”的路线图，不适合作为 CNN 或反向传播的理论依据。
