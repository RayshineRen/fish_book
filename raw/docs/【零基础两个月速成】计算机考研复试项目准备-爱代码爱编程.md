labels文件夹里面是一个个txt文件，txt文件中有很多行数据组成，每一行数据由 序数+四个浮点数表示（这些数据是什么意思呢，解答如下）

第一列是种类的序号，  
第二列是目标的 x 坐标，不是绝对坐标，而是相对坐标，位于整个 x 轴的百分之多少的位置，  
第三列是目标的 y 坐标，不是绝对坐标，而是相对坐标，位于整个 y 轴的百分之多少的位置，  
第四列是目标的 x 轴方向上的长度，不是绝对长度，而是相对长度，占整个 x 轴的百分之多少  
第五列是目标的 y 轴方向上的长度，不是绝对长度，而是相对长度，占整个 y 轴的百分之多少  

不难发现每个jpg文件对应一个txt文件，每个图像都有一个标签

1）–img-size 表示的是训练 / 预测的图片大小，设置为 640。当预测 / 训练时，用户提供的1200 \* 800大小的图片，会先缩放成大小为640的图片，机器会对640大小的图片进行标记，然后再对标记后的图片放大成1200 \* 800大小的图片

2）–conf-thres 表示的置信度，比如说设置为0.25，也就是判断概率大于0.25时，才将这个标记展示出来

3）–iou-thres 进行目标检测时，会画出多个框，如果两个框重叠的部分占比很大，大于这个阈值的话，就合并两个框，只画出一个框；如果两个框重叠部分占比很小，小于这个阈值的话，就分开画出两个框

IoU = 两个图像的交集的面积 比上 两个图象的并集的面积

**if \_\_ name \_\_ == '\_\_ main \_\_ 的含义**

假如说同级目录下，有两个python文件，文件分别为 test.py 和 import\_test.py

test.py 的文件内容为

```python
print('this is one')

if __name__ == '__main__':
    print('this is two')
```

import\_test.py 的文件内容为

```python
import test #没错，只有一个import语句
```

① 如果在命令行执行 test.py （即输入 python test.py），则程序输出  
this is one  
this is two  
② 如果在命令行执行 import\_test.py （即输入 python import\_test.py），则程序输出  
this is one

① 如果是命令行直接执行这个 python 文件，则会执行这个python文件中 if \_\_ name \_\_ == '\_\_ main \_\_ ’ 之后的内容  
② 如果是其他文件执行过程中 import 这个 test.py 文件，则会执行这个 test.py 文件（ **注意！import哪个python文件，就会执行哪个python文件** ），但不会执行这个 test.py 文件中 if name \_\_ == ‘\_\_ main \_\_’ 之后的内容

卷积用来 **提取图像特征** ，池化用来 **降低特征图尺寸**  
**CSP就是用来提高特征提取能力**  
CSP（Cross-Stage Partial Network）指的是 **将特征图分为两个部分，分别进行不同的卷积操作** （既可以对原始图像进行卷积操作提取初级特征，又可以对特征图再进行卷积操作提取高级特征），然后合并在一起，进入下一层  
**PANet用来提升检测效果**  
PANet（Path Aggregation Network） **将来自不同尺度的特征图通过路径聚合网络进行融合** 。引入了一个自底向上的路径，使得从低级别到高级别的特征能够在多次迭代中充分融合。通过自顶向下的路径进行高层特征的细化，这有助于更精确地定位和分类目标。  
**CSP和PANet总的来说就是将图像分成多个部分，分别进行不同的卷积操作，得到不同的特征图，然后将不同的特征图进行聚合，提升了特征提取能力和检测速度**

**CSP** 和 **PANet** 是 **YOLOv5中的网络结构组件** ， **而非一般意义上的“方法”** 。它们在模型中承担具体的特征提取或融合功能

数据增强方法：随机裁剪、旋转、 **色彩抖动**  
随机裁剪说的是随机选定一个位置， **裁剪出一个固定尺寸的区域**  
旋转指的是给图像 **旋转一定的角度**  
色彩抖动指的是 **对图像的颜色进行随机的扰动** （亮度、对比度、饱和度、色调等）  
模型剪枝是指 **去除对最终结果影响较小的神经元、权重值较低的连接**  
模型量化是指 **减少表示权重和激活层输出等这些数据表示的比特数**
